## Bitget Wallet Card 是什么？

Bitget Wallet Card 是 Bitget Wallet 推出的一张加密货币消费卡，底层合作方是 Fiat24。简单理解，就是你在 Bitget Wallet 里开一张虚拟卡，然后用 USDC / USDT 这类资产给卡充值，再拿这张卡去支付海外订阅、线上消费，或者绑定支持的支付场景。

它不是传统银行信用卡，而更接近一张 crypto debit card。你先充值，后消费，卡里有多少钱就花多少钱。

这里要特别提醒一句：**Bitget Wallet Card 和 Fiat24 是联名/合作关系，如果你已经用同一个身份开过 Fiat24，可能无法再次申请 Bitget Wallet Card。** 这个点很关键，别到最后 KYC 都做了才发现重复身份卡住。

## 这张卡有什么特点？

我目前看下来，比较适合中国用户关注的点有这几个：

1. 支持中国大陆用户申请，KYC 时大陆用户一般需要用护照。
2. 虚拟卡开通门槛不高，最低充值一笔资金做激活即可。
3. 支持用来支付 ChatGPT、Gemini、Cursor、Grok 这类海外 AI 订阅。
4. 支持 Base 等网络充值，链上成本相对低。
5. 钱包里还可以做一些海外扫码支付、话费充值等场景。
6. 有实体卡选项，但实体卡不是所有地区用户都能申请，官方页面显示实体卡费用约 49U。

费用这里不能简单写“完全 0 损耗”。我实测体验是很顺，但按官方费用页，APAC 区域目前会看到一些明确费用，例如虚拟卡激活费、crypto-to-fiat 转换费、非美元交易的汇率/外汇费用等。最终以你 App 里申请页面显示为准。

所以我更愿意把它描述成：**适合海外订阅和小额线上消费的低门槛 U 卡工具，而不是绝对 0 成本银行卡。**

## 开通步骤

流程并不复杂，我按普通新用户路径写一遍。

第一步，下载安装并注册 Bitget Wallet。注册时可以填邀请码 `jApNeR`。

第二步，完成 KYC 认证。大陆用户建议直接准备护照，不要等到卡申请环节再临时找资料。

第三步，在钱包里找到 Bitget Wallet Card 入口，申请虚拟卡。

第四步，往卡里充值一笔资金用于激活。页面一般会要求不低于 10U，实际操作时可以充 11U 左右，留一点余量。

第五步，选择充值网络和币种。我的建议是优先看 Base 网络的 USDC，成本低，速度也比较适合这种小额消费场景。当然你也可以用 Binance、Neverless、OKX 等平台转入，关键是确认网络和币种别选错。

第六步，卡激活后，把它添加到你要支付的 AI 服务里，比如 ChatGPT 或 Gemini，然后正常订阅支付。

第七步，支付完成后，回到活动页确认是否计入返现任务。别等到月底才发现自己没有报名活动，或者支付方式不符合要求。
