最近一朋友去上了某个 AI 课程。回来之后找我聊天，一本正经的跟我说他们老师说的「未来发工资都要发 Token 了」。

我淡淡的笑了一下，说这句话本来是黄仁勋说的。他的意思不是说就不给员工发钱了，而是说在工资的薪酬体系中，新增了 Token，作为除了工资、奖金、股权之后的「第四种薪酬」。

然后，我也发挥了一下想象力。

如果有一天真的工资都用 Token 发，那不得全部都是 Agent 员工。因为，只有 Agent 工作的时候需要通过计算消耗 Token。所以，Agent 只要有了 Token，就能继续干活了。

那再假如，如果有一天我们人类的世界也用 Token 来做一般等价物取代现在的货币，会怎么样呢？

真到这个时候，我想这个世界就已经被 AI 统治了，我们人类要生存，就要去跟 AI 打工赚 Token 了。如果那一天来临，真不知道人类能干什么。

听到这里，我的朋友破口大骂「AI 是个什么玩意」。

哈哈哈
