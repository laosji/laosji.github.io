# 手续费哪家最低？

最近，股票可算是跌惨了。于是就把散落在各个账户的零钱凑了凑，正好也踩了一些坑。今天就来给大家分享一下，我常用的这些平台的手续费，以及还有哪些羊毛可以薅。

## Remitly

先来分享个羊毛 Remitly。从美国转到英国，首笔金额大于 50 USD，可立减 25 USD（之前是 30USD，最近下调了），并且免手续费，还可以享受特别的优惠汇率。

我之前注册了 3 个账号：英国 → 中国、美国 → 英国、英国 → 美国。现在到中国已经没有羊毛了。

从美国转英国时，我选错了付款卡，余额不足导致被要求补材料并电话确认。我打给 Remitly 客服后，小哥应该是印度的，非常热情。一听我是中国人直接说“你好”。

沟通半小时后，他说我注册了多个账号（Remitly 不允许），但可以帮我关闭其他账号，只保留一个，并让我保证不再注册更多账号。随后他转接另一个客服核对我的身份和付款失败原因，还拉了一个中文翻译。最后，汇款顺利通过。

羊毛邀请链接在这里（还有 11 个名额）：

https://remit.ly/2ziaul1q

## 美区 PayPal 和华美

使用 ACH 方式从 PayPal 提现到华美银行：

- **手续费：免费**
- **到账时间：1 个工作日**

## iFastGB → iFastGM → 港卡

- iFastGB → iFastGM：**免费 + 秒到账**
- iFastGM 出金至港卡：**必须先兑换成港币**

⚠️ 如果你直接从 iFastGM 以英镑汇到港卡，会被收 **35 英镑手续费**，这个坑已经替大家踩过了。

注册 iFAST，入金 500 英镑并保持 3 个月，可获得 5 英镑奖励。

## 汇丰银行

英国汇丰 → 香港汇丰（同名账户）转账：

- **手续费：全免**
- 是常用的 **英 → 港 免费通道**

## 渣打银行

非优先理财用户：  
- 内地 → 香港（美元）每笔 **120 USD 手续费**

因为内地渣打转美元必须去柜台，所以直接从内地转到香港。后来我把账户等级降级了。优先理财客户则是免费。

## 港币回国

港币回国我用熊猫速汇。

首笔汇款免手续费：  
- 需要用 **从未注册过熊猫的新手机号**（非本人也可以）  
- 适用于外币 → 中国境内
