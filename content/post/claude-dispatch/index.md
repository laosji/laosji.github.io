## Claude龙虾也能远程安排电脑自己工作了

Claude 推出了重磅更新 **Coworking on the Go**，让你的手机变成对讲机，随时随地给电脑上的 Claude 派活。

## 三大核心能力

### 📱 Dispatch — 手机派任务
手机就像对讲机，发消息给电脑上的 Claude，它自动操作本地文件、Slack、邮件等。

### 💻 Remote Control — 远程写代码
在手机或任意浏览器上远程控制 Claude Code 终端，开发不中断。

### ⏰ 定时任务 — 自动执行
用 /schedule 设定每天、每小时自动执行任务，AI 帮你值班。

### 🎙️ 语音模式 — 按住说话
终端里按空格键像对讲机一样语音指令，说完 Claude 自动执行。

## 真实使用场景

- 🚇 **通勤路上派活**：地铁上用手机发消息"帮我重构登录模块"，到公司打开电脑代码已经写好了
- ☕ **喝咖啡时查进度**：手机上随时查看 Claude 的工作进度，追加需求或微调方向
- 🌙 **睡前定时任务**：设置每天早上自动拉取数据、生成日报，醒来就能看结果
- 🏖️ **度假也能救火**：手机语音说一句"查一下线上报错日志"，Claude 远程在你电脑上执行
