## Give Claude, get more Claude

Claude 最近上线了一个「访客通行证（Guest Pass）」的邀请玩法，口号叫 **Give Claude, get more Claude**——把 Claude 送给朋友，自己也能拿到更多 Claude。

简单说：你可以送朋友 **7 天免费的 Claude Pro**，让他完整体验 **Claude Code** 和 **Cowork**；如果朋友用下来喜欢、转成付费订阅，你就能拿到 **$10 的使用额度（usage credits）** 继续撸 Claude。

## 双方各能得到什么

### 🎁 朋友（被邀请人）

- **7 天免费 Claude Pro**，完整功能
- 同时可以用 **Claude Code**（命令行写代码）和 **Cowork**（任务协作）
- 引导式安装，注册流程简单

### 💰 你（邀请人）

- 朋友转成付费订阅后，你获得 **$10 使用额度**
- 每个符合条件的账户有**限量**通行证（界面里显示 3 张），先到先得

## 谁能送通行证？

- 仅限 **Pro 和 Max** 订阅用户可以发放访客通行证
- 在启动 Claude Code 会话或 Cowork 任务时会收到提示
- 获取链接的方式：Claude Code 里输入 `/passes` 命令、或在设置 / 桌面 App 里复制专属邀请链接

## 朋友领取的条件与注意事项（重要）

- 必须是 **从未付费过** 的 Claude 新用户（没有订阅历史）
- 激活试用 **需要先填写支付信息**
- 7 天试用结束后会**自动转为付费 Pro 订阅**，不想续费的话记得**到期前取消**
- 通行证会过期，按**先到先得**发放；一旦被领取或过期，就不能再发

## 关于 $10 额度的规则

- 这是**使用额度**，**没有现金价值、不可转让**
- 自发放起 **一年后过期**
- 如果你**降级回免费版**，额度会失效
